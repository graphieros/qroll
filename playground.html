Smooth page slider

All you need is one ID, and a dash of css classes



<div id="qroll-parent" class="qroll-transition-500 qroll-nav">

    <div id="anyIdWorks1" class="any-class-you-want">
      <h1> Title </h1>
    </div>

    <div id="anyIdWorks2" class="any-class-you-want">
      <h2> Slide </h2>
    </div>

</div>

Nested carousels

Nested carousels is as easy as adding a few css classes to any div included in the main parent. You can also use a data attribute to customlize the progress bar style:



<div id="qroll-parent" class="qroll-transition-500 qroll-nav">
    <div class="any-class-you-want">
        <div class="qroll-nested-carousel qroll-progress qroll-transition-300 qroll-loop" data-progress-css="background: linear-gradient(to right, transparent, black)">
          <div> Nested Carousel slide 1 </div>
          <div> Nested Carousel slide 2 </div>
          <div> Nested Carousel slide 3 </div>
        </div>
    </div>
</div>

Native behaviors

Scrolling inside this div must not trigger page sliding.
