roll

A friendly full page scroll library

npm i qroll scroll

Easy

With a few drops of css classes


<div id="qroll-parent" class="qroll-main qroll-loop qroll-tooltip">

    <div id="anyIdWorks1" class="any-class-you-want">
      <h1> Title </h1>
    </div>

    <div id="anyIdWorks2" class="any-class-you-want">
      <h2> Slide </h2>
    </div>

</div>

It also works sideways

Just a few css classes
to create horizontal slides


<div id="qroll-parent" class="qroll-main qroll-loop qroll-tooltip">

    <div class="qroll-carousel qroll-loop qroll-tooltip">
      <div> Horizontal slide 1 </div>
      <div> Horizontal slide 2 </div>
      <div> Horizontal slide 3 </div>
    </div>

    <div id="myId2">
      <h2> Vertical slide 2 </h2>
    </div>

</div>

Qroll allows you to create carousel components. Anywhere inside a slide, just drop a css class on a div and voil√†.

      <h1> Slide 1 </h1>

</div>
